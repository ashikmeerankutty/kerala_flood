# node_starter
## How to start
### Install node modules
`npm install`
### Create .env file in root location of project
`touch .env`

    PORT=3000
    NODE_ENV=development
    DATABASE=mongodb://localhost:port/database-name

### Start project
`npm start`

### Running in developement with nodemon
`npm run dev`

### Running test
`npm run test`# kerala_flood
